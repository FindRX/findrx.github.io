Sqrt(x)

Implement int sqrt(int x).

Compute and return the square root of x.

C Solution:

int mySqrt(int x) {
    if (x < 1) return 0;
    int l = 1, r = x;
    while (l + 1 < r) {
        int m = l + ((r - l) >> 1);
        if (m == x / m) return m;
        if (m < x / m) {
            l = m;
            r = x / m;
        }
        else {
            l = x / m;
            r = m;
        }
    }
    return l;
}
