Can Place Flowers

Suppose you have a long flowerbed in which some of the plots are planted and some are not. However, flowers cannot be planted in adjacent plots - they would compete for water and both would die.

Given a flowerbed (represented as an array containing 0 and 1, where 0 means empty and 1 means not empty), and a number n, return if n new flowers can be planted in it without violating the no-adjacent-flowers rule.

Example 1:

  • Input: flowerbed = [1,0,0,0,1], n = 1
  • Output: True

Example 2:

  • Input: flowerbed = [1,0,0,0,1], n = 2
  • Output: False

Note:

  • The input array won't violate no-adjacent-flowers rule.
  • The input array size is in the range of [1, 20000].
  • n is a non-negative integer which won't exceed the input array size.

C Solution 1:

bool canPlaceFlowers(int* flowerbed, int flowerbedSize, int n) {
    int i, res = 0;
    for (i = 0; i < flowerbedSize; i++) {
        if (flowerbed[i]) continue;
        if ((!i || !flowerbed[i - 1]) && (i + 1 == flowerbedSize || !flowerbed[i + 1])) {
            res++;
            flowerbed[i] = 1;
        }
    }
    return n <= res;
}

C Solution 2:

bool canPlaceFlowers(int* flowerbed, int flowerbedSize, int n) {
    if (!n) return true;
    if (flowerbedSize < 1) return false;
    if (flowerbedSize == 1) return !flowerbed[0];
    int res = 0;
    if (!flowerbed[0] && !flowerbed[1]) {
        res++;
        flowerbed[0] = 1;
    }
    int i;
    for (i = 1; i < flowerbedSize - 1; i++) {
        if (flowerbed[i]) continue;
        if (!flowerbed[i - 1] && !flowerbed[i + 1]) {
            flowerbed[i] = 1;
            res++;
        }
    }
    if (!flowerbed[flowerbedSize - 1] && !flowerbed[flowerbedSize - 2]) {
        res++;
    }
    return n <= res;
}

Summary:

  • Don't think too much about performance.
  • With extra comparation, code are much more concise.
