204. Count Primes

Count the number of prime numbers less than a non-negative number, n.

C Solution:

int countPrimes(int n) {
    if (n < 3) return 0;
    char *flag = calloc(n, 1);
    int i, res = n - 2;
    for (i = 2; i * i < n; i++) {
        if (flag[i]) continue;
        int j;
        for (j = i * i; j < n; j += i) {
            if (!flag[j]) {
                res--;
                flag[j] = 1;
            }
        }
    }
    return res;
}

Summary:

  1. '1' is not a prime.
  2. This is called Sieve of Eratosthenes, so beautiful.
  3. 32ms, 98.1%
