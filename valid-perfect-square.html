Valid Perfect Square

Given a positive integer num, write a function which returns True if num is a perfect square else False.

Note: Do not use any built-in library function such as sqrt.

Example 1:

  • Input: 16
  • Returns: True

Example 2:

  • Input: 14
  • Returns: False

Python Solution 1:

class Solution(object):
    def isPerfectSquare(self, num):
        """
        :type num: int
        :rtype: bool
        """
        l, r = 0, num
        while l <= r:
            m = l + (r - l) // 2
            print m
            multi = m * m
            if multi == num: return True
            if multi > num:
                r = m - 1
            else:
                l = m + 1
        return False

Python Solution 2:

class Solution(object):
    def isPerfectSquare(self, num):
        """
        :type num: int
        :rtype: bool
        """
        i = 1;
        while num > 0:
            num -= i
            i += 2
        return not num

Python Solution 3:

class Solution(object):
    def isPerfectSquare(self, num):
        """
        :type num: int
        :rtype: bool
        """
        x = num
        while x * x > num:
            x = (x + num // x) >> 1
        return x * x == num
