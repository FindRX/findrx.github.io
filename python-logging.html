Python: Logging

Logging is performed by calling methods on instances of the Logger class.

Each instance has a name, and they are conceptually arranged in a namespace hierarchy using dots (periods) as separators.

The root of the hierarchy of loggers is called the root logger.

  • The root logger’s name is printed as ‘root’ in the logged output.

By default:

  • the output destination is sys.stderr.
  • The default format set by logging.basicConfig() for messages is:
    • severity:logger name:message
  • The level of the root logger is WARNING.

If a level is not explicitly set on a logger, the level of its parent is used instead as its effective level.

Child loggers propagate messages up to the handlers associated with their ancestor loggers.
