Excel Sheet Column Number

Related to previous question 168. Excel Sheet Column Title

Given a column title as appear in an Excel sheet, return its corresponding column number.

Example:

  • A -> 1
  • B -> 2
  • C -> 3
  • ...
  • Z -> 26
  • AA -> 27
  • AB -> 28

C Solution:

int titleToNumber(char* s) {
    int res = 0;
    for (; *s; s++) {
        res = res * 26 + *s - 'A' + 1;
    }
    return res;
}

Python Solution 1:

class Solution(object):
    def titleToNumber(self, s):
        """
        :type s: str
        :rtype: int
        """
        sum = 0
        for c in s:
            sum = sum * 26 + ord(c) - ord('A') + 1
        return sum

Python Solution 2:

class Solution(object):
    def titleToNumber(self, s):
        """
        :type s: str
        :rtype: int
        """
        return reduce(lambda x, y: x * 26 + y, [ord(c) - ord('A') + 1 for c in s])
